# Roadmap

## Milestone 1: v1 Static Guide

Ship the first public version of GEO Basics as a bilingual static site.

### Features

- `geo-basics-guide`: beginner guide, diagrams, code snippets, checklists, references, and bilingual pages.

### Done When

- English and pt-BR pages are available as static GitHub Pages paths.
- All diagrams render as SVG and have accessible alt text.
- Local validation checks pass.
- Browser verification confirms desktop and mobile layouts are usable.

## Future Ideas

- Add Spanish translation.
- Add printable one-page checklist.
- Add examples by content type: SaaS, local business, ecommerce, blog, documentation.
- Add a small JSON-LD generator for Article, FAQPage, HowTo, and Organization examples.
- Add automated link checking in CI after the repository remote is configured.
